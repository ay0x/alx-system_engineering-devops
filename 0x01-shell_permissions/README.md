This folder contains shell scripts that manipulate permissions.

0-iam_betty: Script that switches the current user to another user

1-who_am_i: Script that prints the effective username of the current user.

2-groups: script that prints all the groups the current user is part of.

3-new_owner: Script that changes the owner of a file to another owner

4-empty: Script that creates an empty file

5-execute: Script that adds execute permission to the owner of the file

6-multiple_permissions: Script that adds execute permission to the owner and the group owner, and read permission to other users, to a file

7-everybody: Script that adds execution permission to the owner, the group owner and the other users, to a file without using commas in the mode

8-James_Bond: Script that sets the 007 permission to a file

9-John_Doe: Setting mode of a file

10-mirror_permissions: Script that sets the mode of a file from another file’s mode

12-directory_permissions: Script that creates a file and sets a permission to that file

13-change_group: Script that changes the group owner of a file

100-change_owner_and_group: Script to changes the owner and group owner of the files and sub directories of a working directory

101-symbolic_link_permissions: Script that changes the owner and group owner of a symbolic link

102-if_only: Script that changes the owner of a file to another owner if the other owner is owned by a particular owner
