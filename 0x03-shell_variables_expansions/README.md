This is a README.md for Shell, init files, variables and expansions projects. Enjoy!

0-alias: Script that creates an alias

1-hello_you: Script prints hello user where user is the current Linux user

2-path: The script adds  /action to the PATH variable

3-paths: The script counts the number of directories in the PATH variable

4-global_variables: The script lists the environment variables

5-local_variables: The script lists the local variables

6-create_local_variable: Script to create local variable

7-create_global_variable: Script to create global variable

8-true_knowledge: Script for adding a number to an integer variable

9-divide_and_rule: Script for dividing integer variables

10-love_exponent_breath: Script to create exponents between integer variable

11-binary_to_decimal: Script for converting base 2 to base 10

12-combinations: Script for creating possible combination of two alphabets

13-print_float: Script for adding floats to integers

100-decimal_to_hexadecimal: Script for converting base 10 to base 16

101-rot13: Script for converting texts to ROT13 encryption
